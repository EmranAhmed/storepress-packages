# StorePress JavaScript Packages

- Check docs: https://emranahmed.github.io/storepress-packages/

## Init

- `npx lerna init --packages="packages/*" --independent`


## Storybook

- `npm run build`
- `npm run storybook`

## Build

- `npm run build`

## Publish

- `npx lerna publish from-package --no-private`
